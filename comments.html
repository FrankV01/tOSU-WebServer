Welcome

Welcome to my Joke Server project. I hope you have a great visit. This document contains some quick notes on building and running the programs.

Compiling

A build can be accomplished one of two ways. The first is manually running javac the second is to use ant. The two methods are outlined below.

javac

The are three main files that must be compiled. Any dependencies should be compiled inherently.

The files are:

  • JokeClient.java
  • JokeClientAdmin.java
  • JokeServer.java

To compile the files, simply execute javac for each source file. The following are examples.

  • javac file.java

Ant [Optional]

If you desire, you can try to use ant to build the files. Using ant saves some typing but it truly isn't required.

I've provided a build file called build.xml. Simply point your command window to the directory you unzipped the files at. And then execute ant. A basic build will be done and the resulting class files will be stored in "build".

Further ant build options can be explored via the ant -projecthelp command. This will simply list all build targets but the default is the most commonly used one.
